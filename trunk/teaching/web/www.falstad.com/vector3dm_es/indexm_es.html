Este applet java es una demostración magnetostática, para ver el campo magnético en una diversidad de situaciones. Puede Ud. seleccionar los campos y ver como partículas se mueven si el campo fuera uno de velocidades en un fluido (las partículas se mueven a lo largo de las líneas de campo), o ver el campo de fuerzas real (las partículas se mueven como si fueran pequeños imanes). Esto ayuda a visualizar el campo. También puede Ud. ver el potencial vector (A).

En el menú de arriba a la derecha están las opciones de campo para elegir.
