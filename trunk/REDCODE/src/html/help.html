Redcode assembler and simulator

Assembly code

Enter redcode assembly code into the Editor pane. If you want to save/load code you can cut and paste (control-C and control-V).

The assembler recognises labels and simple expressions. For example 

    X:  DAT 0
        DAT 0
    Y:  DAT 0
        MOV #X-Y  Y
will compile into 
        DAT 0
        DAT 0
        DAT 0
        MOV #-2  -1
The assembler understands character constants 
        DAT 'H'
        CMP -1 #'H'
will compile into ascii code for 'H' 
        DAT 72
        CMP -1 #72

Compiling and loading

The compile/load button will attempt to compile the code and load it into the simulator (right panel). If there are any errors they should be reported and highted.
