Instruction Notes
MOV A B Move(copy) A to B
ADD A B Add A to B
SUB A B Subtract A from B
JMP A Jump to A (PC=A)
JMZ A B if B is zero jump to A
JGZ A B if B greater than zero Jump to A (PC=A)
DJZ A B subtract 1 from B, if B is now zero jump to A (PC=A)
CMP A B if A is not equal to B skip next instruction (PC=PC+2)
IN A get a character from input and out into A
OUT A transfer A to the output
HLT Terminate execution
DAT B Directive to initialise memory of the B field with a value.
Character Mode
@ Indirect
# Immediate
direct
