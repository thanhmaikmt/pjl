Redcode tutorial exercises

Basic

Add some numbers 

    first:  DAT 3
            DAT 5
    last:   DAT 7
    result: DAT 0
After execution result should contain the sum of the numbers from first to last.

Multiplication 

    a:    DAT 6
    b:    DAT 5
    res:  DAT 0
After execution res should contain the product a times b.

Division 

    num:  DAT 21
    den:  DAT 7
    div:  DAT 0
    rem:  DAT 0
After execution div should contain the integer division num/den whilst rem should hold the remainder.

Reverse a list 

    first:  DAT 3
            DAT 5
    last:   DAT 7
After execution the list should be reversed (e.g. 7 5 3). Your code should work for any size of list.

Input/output

Greeting

Write a program that allows the user to enter his/her name (say "fred") then outputs "Hello fred"

Hello

Write a program that responds "Hello" if the user enters "pass".

Advanced

Sorting 

    first:  DAT 21
            DAT 7
            DAT 13
    last:   DAT 10
After execution the list should be sorted into asscending order.

Ascii code

Write a program reads a character and outputs it's ascii code. For example 'H' should print out "72"

Factorial

Write a program that calculates the factorial of a number.
